# TICTACTOE ( Python Tkinter )
 Simple game of tictactoe that can be played against another user or unbeatable computer player made using tkinter library. App utilizes the minimax algorithm to choose the best possible move to play based on simulated a play by play.
 
#### Source code: tictactoe.py
#### Application: tictactoe.exe

## How to run
1. Sownload _internal folder and tictactoe.exe
2. Make sure application is one directory out of _internal like shown below.
  
   ![Screenshot 2024-09-27 001950](https://github.com/user-attachments/assets/6d2500a5-559a-4acc-bdf0-477a5657a804)
3. Launch game and play.
